# Task 8: Simple Sales Dashboard

This project is a Tableau-based dashboard designed to analyze sales performance from the Sample Superstore dataset.

## Objective
To visualize and evaluate sales trends by product, region, and month using Tableau charts and a combined interactive dashboard.

## Contents
- `SampleSuperstore.csv` – Dataset used
- `Simple_Sales_Dashboard.twbx` – Tableau workbook
- `/screenshots/` – Screenshots of individual charts and dashboard
- `Simple_Sales_Dashboard_Insights_PPT_by_Jyoti_Chahal.pptx` – Insight presentation

## Visuals
- Monthly Sales Trend
- Sales by Region
- Top 10 Products by Sales
- Combined Simple Sales Dashboard

## Tools Used
- Tableau Public
- PowerPoint
- GitHub
